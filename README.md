# 💎 Gemstone Recommendation App

A full-stack MERN application that recommends gemstones based on user inputs such as preferences and custom rules. It includes authentication, personalized recommendations, and a modern responsive UI.

---

## 🚀 Live Demo

👉  https://gemstone-recommendation-app-1.onrender.com  

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- JavaScript (ES6+)
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt.js
- CORS
- dotenv

### Deployment
- Render (Frontend + Backend)
- MongoDB Atlas (Database)
- GitHub (Version Control)

---

## ✨ Features

- User Signup & Login system
- Secure authentication using JWT
- Personalized gemstone recommendations
- Rule-based recommendation logic
- Responsive UI design
- REST API integration
- MongoDB cloud database storage
- Clean and simple user experience
